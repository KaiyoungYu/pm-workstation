- **[PRD-example.md](./PRD-example.md)** - PRD 标准模板（Agent 编写产品设计文档的基准）

## 使用指南

开始编写需求文档前，请先阅读 `doc-types.md` 和 `file-structure.md`，了解文档的组织方式和命名约定。

## 文档生命周期

本项目采用三阶段文档工作流，详见 [../三个阶段.md](../三个阶段.md)：

1. **阶段一：需求分析** - 产出 BRD（需求分析文档）
2. **阶段二：产品原型设计** - 产出 PRD（原型设计说明）+ Prototype
3. **阶段三：开发需求编写** - 产出 SRD（开发需求文档）
