# SRD 模板示例（Specification Requirements Document）

> **用途**：本文件为 `knowledge/02-requirements/REQ_*/SRD.md` 的标准模板。Agent 在编写或评审 SRD 时，应以此模板为基准，确保技术规格无歧义、可直接驱动 Coding Agent 编码。
>
> **核心原则**：SRD 回答 "How — 怎么做？"，是 BRD 和 PRD 的技术精确化。它定义接口路径、参数命名、数据类型、校验规则、状态机、测试规格等。**SRD 中不应引入 BRD 中未确认的需求，也不应改变 PRD 中已确定的功能逻辑**。
